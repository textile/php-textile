h1. List syntax

* List items appear on consecutive lines.
* Lists may be nested by repeating the list item identifier. 
* There is no dot after the identifier.
* Attributes may be applied immediately after the identifier.
* Attributes declared with the first item are applied to the list as a whole.
  
###. The Textile synax examples below are enclosed in <pre> blocks. 
     This is so the Textile markup can be seen in the HTML output
     from this file.

h2. Mixed list

An ordered list can be nested in a bulleted list item, and vice versa:

pre. 
# Choose a platform
** PHP
### Version 4
### Version 5
** Ruby
# Choose a markup language
** Textile
** Markdown

    <li>Choose a platform
